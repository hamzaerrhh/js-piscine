# JavaScript Piscine – Zone01

## Overview

This repository documents my journey through the **JavaScript Piscine** at **Zone01**, an intensive project-based program designed to build strong problem-solving skills and a deep understanding of JavaScript. Rather than following traditional lectures, the Piscine focuses on hands-on exercises and progressively more challenging projects.

I successfully completed the entire program, finishing every quest and the final raid by applying the concepts learned throughout the curriculum.
